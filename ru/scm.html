Обзор

Данный проект использует Git для управления исходным кодом. Информацию по использованию Git можно найти по адресу: https://git-scm.com/documentation.

Web-Доступ

Ниже приведена ссылка для web-доступа к репозиторию исходного кода проекта:

https://github.com/IFML2/ifml3

Анонимный Доступ

Рабочая копия из Git репозитория может быть получена анонимно с помощью следующей команды (см. https://git-scm.com/docs/git-clone):

$ git clone https://github.com/IFML2/ifml3.git

Доступ для Разработчиков

Только разработчики проекта могут получить доступ к дереву Git, используя данный метод (см. https://git-scm.com/docs/git-clone).

$ git clone git@github.com:IFML2/ifml3.git

Доступ через брандмауэр (firewall)

Для получения дополнительной информации о доступе к репозиторию через брандмауэр (firewall) обратитесь к документации конкретной системы управления версиями (SCM).
